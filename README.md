🚗 Vehicle Insurance System

A full-stack web-based Automobile Insurance Management System developed to modernize vehicle insurance operations by managing policies, claims, premium payments, customer information, and proposal approvals in a secure and efficient way.

📌 Project Overview

The Vehicle Insurance System allows customers to:

Register and log in securely
Apply for vehicle insurance policies
Track proposal and policy status
Make premium payments
Download policy documents
Raise and track insurance claims

The system also provides an officer/admin dashboard for:

Reviewing proposals
Approving/rejecting applications
Managing policies and claims
Generating quotes
Processing claim payments
